Mes Projets

Projet PUNTOS


Description du projet "Puntos"


Objectif du projet

Puntos est un jeu web interactif qui combine les mécaniques classiques de Puissance 4 avec une dimension stratégique unique. L'objectif est de proposer une expérience ludique et compétitive où les joueurs peuvent planifier leurs actions tout en s'amusant.


Technologies utilisées


Front-End : HTML, CSS, JavaScript.

Back-End : Node.js, Express.js, MongoDB.

Format de données : JSON.

Fonctionnalités principales


Authentification sécurisée des joueurs.

Gestion des scores et des parties via une base de données.

Interface utilisateur responsive.

Communication fluide entre le front-end et le back-end grâce à une API REST.

Mon rôle

J'ai développé le système d'authentification des utilisateurs, conçu l'interface utilisateur en HTML/CSS/JavaScript, et configuré l'API REST avec Express.js pour gérer les données des parties et des joueurs.
